# README

## 拡散モデルと多様なデータセット

このプロジェクトは、拡散モデルの構築をテーマに、シンプルな分布から複雑なデータセットに至るまでのさまざまなデータを使用して、その性能と挙動を解析します。それぞれのデータセットに対応するJupyterノートブックを用意し、逆拡散過程をニューラルネットワークで学習し、その適用例を探求します。

---

## プロジェクト構成

以下のノートブックとデータセットが含まれています。

### ノートブック一覧

| **No.** | **Notebook**                         | **データセット/タスク**                   | **説明**                                                                 |
|---------|--------------------------------------|-------------------------------------------|--------------------------------------------------------------------------|
| 01      | `01_1D_Normal_Distribution.ipynb`    | 1次元正規分布                             | 基本的な1次元ガウス分布で拡散モデルをテスト。                             |
| 02      | `02_2D_Normal_Distribution.ipynb`    | 2次元正規分布                             | 2次元の空間データで拡散モデルを適用し、ノイズ除去を実験。                 |
| 03      | `03_Gaussian_Mixture_Model.ipynb`    | ガウス混合モデル（GMM）                   | 複数のモードを持つデータ（マルチモーダル）を処理する能力を評価。          |
| 04      | `04_MultiDimensional_Normal_Distribution.ipynb` | 多次元正規分布                  | 高次元データを用いて拡散プロセスのスケーラビリティを検証。                |
| 05      | `05_MNIST_Image_Denoising.ipynb`     | MNIST（手書き数字画像）                   | 拡散モデルを使用して手書き数字画像のノイズを除去。                         |
| 06      | `06_Spiral_Pattern.ipynb`            | スパイラルパターン                        | 非線形なスパイラルデータを用い、構造化されたデータへの適用をテスト。       |
| 07      | `07_CIFAR10_Image_Denoising.ipynb`   | CIFAR-10（小型画像データセット）          | 小型のRGB画像に拡散モデルを適用し、ノイズ除去を検証。                     |
| 08      | `08_Audio_TimeSeries_Denoising.ipynb`| 音声時系列データ                          | 1次元の時系列データを用い、ノイズ除去を実験。                             |
| 09      | `09_Text_Embedding_Denoising.ipynb`  | テキスト埋め込み                          | 自然言語データ（単語や文の埋め込み）のノイズ除去を試行。                   |
| 10      | `10_Game_State_Space_Reconstruction.ipynb` | ゲーム状態空間データ               | ゲーム状態のノイズを除去し、元の表現を再構築。                            |

## 実行方法

1. **依存関係のインストール**  
   プロジェクトを実行する前に必要なPythonライブラリをインストールしてください。以下のコマンドを使用します：
   ```bash
   pip install -r requirements.txt

2. **データセットの準備**  
   各ノートブックに記載されたデータセットの準備手順を確認してください。以下に主なデータセットの準備方法を示します：

   - **MNIST/CIFAR-10**:  
     これらのデータセットはノートブック実行時に自動的にダウンロードされます。特別な準備は不要です。

   - **スパイラルパターンや正規分布**:  
     ノートブック内で生成コードが実行されるため、事前の準備は不要です。

   - **音声データやゲーム状態空間**:  
     必要に応じて、外部のデータセットを事前にダウンロードしてください（リンクや手順は該当ノートブック内に記載）。

3. **ノートブックの実行**  
   プロジェクトディレクトリに移動し、Jupyter Notebook（またはJupyter Lab）を起動します：
   ```bash
   jupyter notebook
